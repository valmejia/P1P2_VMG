# P1P2_VMG

## Descripción

Aplicación Android nativa desarrollada en **Java** con **Android Studio** que implementa un formulario simple de una sola pantalla con las siguientes características:

- Un **campo de texto** para ingresar el nombre del usuario
- Un **botón** para enviar el formulario
- Una **alerta** que muestra un mensaje de bienvenida personalizado con el nombre ingresado

> **Ejemplo:** Si el usuario escribe "Juan" y presiona el botón, aparece una alerta que dice: *"¡Bienvenido, Juan!"*

## Cómo ejecutar el proyecto
1. Clona el repositorio:
   
   ```bash
   git clone https://github.com/valmejia/P1P2_VMG.git

2. Abre Android Studio.
3. Espera a que se sincronicen las dependencias (Gradle).
4. Conecta un dispositivo físico o inicia un emulador.
5. Haz clic en Run.
